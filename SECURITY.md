# Security Policy

## Supported Versions

Currently supported versions with security updates:

| Version | Supported          |
|---------|-------------------|
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please send an email to security@superinstance.dev.

Please include:

* Description of the vulnerability
* Steps to reproduce
* Potential impact
* Suggested fix (if known)

We will:

* Acknowledge receipt within 48 hours
* Provide regular updates on our progress
* Notify you when we've deployed a fix
* Credit you in the release notes (if desired)

Please do NOT:

* Publicly disclose the vulnerability before it's fixed
* Attempt to exploit the vulnerability in production
* Use the vulnerability for any malicious purpose
